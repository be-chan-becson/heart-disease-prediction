# Heart Disease Prediction

Machine learning project to predict heart disease using clinical data.

## Model
Logistic Regression

Accuracy: ~88%

## Workflow
- Data Cleaning
- Exploratory Data Analysis
- Correlation Heatmap
- Logistic Regression Model
- Confusion Matrix
- ROC Curve

## Dataset
UCI Heart Disease Dataset
